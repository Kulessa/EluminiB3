# Teste Elumini B3

## Como fazer para executar o projeto?

1. Clone o repositório
2. Abrir a solução (CristianRichardKulessa.Elumini.B3.sln) no Visual Studio 2022 
3. Compilar a solução
    3.1. Acesse o menu Build -> Build Solution ou utilize o atalho de teclado F6
4. Executar testes unitários
    4.1. Acesse o menu Test -> Run All Tests ou utilize o atalho de teclado Ctrl + R, A
5. Executar o projeto
    5.1. Configurar Multiple Startup Projects
        5.1.1. Acesse o menu Project -> Set Startup Projects
        5.1.2. Selecione a opção `Multiple startup projects`
        5.1.3. Selecione a opção `Start` para os projetos `CristianRichardKulessa.Elumini.B3.Server` e `cristianrichardkulessa.elumini.b3.client`
    5.2. OU Acesse o menu Debug -> Start Debugging ou utilize o atalho de teclado F5
    5.3. OU Acesse o menu Debug -> Start Without Debugging ou utilize o atalho de teclado Ctrl + F5
